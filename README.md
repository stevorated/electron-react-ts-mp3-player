# Pleya :stew:

An Electron React mp3 manager app using typescript.

## Basic Instructions

Before everything:

```
git clone ...
yarn
```

Starting the app in dev mode (HMR available in react part of the app):

```
yarn dev
```

Starting the app in:

```
yarn start
```

Build the app in:

```
yarn build
```
